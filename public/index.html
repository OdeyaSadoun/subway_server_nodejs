T r a v e l s - R o u t e s

Some "GET" requests you can get information about the API, By adding these queries you can see the edit

  • Per-Page : you can select the amount of obj per one page.
  • Page : you can select the num of pages you want.
  • Sort : you can select how to make your sort - by Price / ID / Name.
  • Reverse : you can choose the sort dirction.

D o m a i n : https://subway-server-nodejs.onrender.com

#Method URL Body Description
GET /allTravelsList none You need an admin token for this query.
Get a list of all travels in the system from all the users, will present 10 results per page.
GET /travelsUserList none You need a token for this query.
Get a list of all your travels, will present 10 results per page.
GET /travels/search?price=20 none You need a token for this query.
search for your travels by searching follwing price after the "price=", it will present 10 results per page.
GET /travels/search?payment_type=day ticket none You need a token for this query.
search for your travels by searching follwing payment type of ticket after the "payment_type=", it will present 10 results per page, it will present 10 results per page that sort by ascending order.
GET /travels/searchByPriceRange?min=10&max=20 none You need a token for this query.
Get a list of your travels by min & max price , it will present 10 results per page that sort by ascending order.
